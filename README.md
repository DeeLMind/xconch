# 神奇海螺

***

## 简介

是时候该结束开发板裸奔的时代了。采用堆叠设计，拥有漂亮外观及超强扩展性。一款完全开源的结构及通用硬件扩展平台，旨在大家能更欢乐的DIY。

![xconch-introduced ](https://github.com/xboot/xconch/blob/master/documents/images/xconch-introduced.png)
